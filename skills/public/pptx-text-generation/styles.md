# Style Presets

When the user requests a specific style, or when choosing one for their topic, apply the **complete design system** below — not just the colors. Each preset defines palette, typography, shape language, visual motif, background strategy, and image-generation modifiers.

**How to use**: Pick the style that best matches the user's topic or request. Apply ALL properties consistently across every slide. Don't cherry-pick — the cohesion is what makes a style work.

**If the user doesn't specify a style**: Suggest 2–3 that fit their topic and let them choose.

## 🎨 Custom / Freeform

When no preset fits the topic — or when the content deserves something truly unique — **design a bespoke style from scratch**. Don't force a jazz presentation into "Dark Professional" or a medical deck into "Nordic Research" just because they're the closest match.

### When to go custom

- The topic has a strong visual identity that no preset captures (e.g., jazz, space exploration, culinary arts, medicine, sports)
- The user explicitly asks for something original or different
- You feel a preset would make the deck look generic for this particular content
- The presentation mixes domains (e.g., "art meets engineering") and no single style covers it

### How to design a custom style

Follow the same structure as the presets. Define ALL of these properties — the structure is what makes a style cohesive, not which preset it came from:

| Property | What to define |
|----------|---------------|
| **Primary** | The dominant background/brand color (60-70% visual weight) |
| **Secondary** | Supporting tone for contrast and depth |
| **Accent** | A single pop color for emphasis, callouts, key elements |
| **Text** | High-contrast text colors for both light and dark backgrounds |
| **Header font** | A font with personality that reflects the topic |
| **Body font** | A clean, readable companion font |
| **Shapes** | Shape language — sharp or rounded? Bordered or filled? Thick or thin? |
| **Motif** | The repeating visual element that ties all slides together |
| **Backgrounds** | Which slides get which treatment (solid, image, dark, light) |
| **Image prompts** | A modifier phrase to append to all `generate_image` calls for visual consistency |

### Color design tips

- **Draw from the subject matter** — a jazz deck might use smoky midnight blues and warm brass golds; a medical deck might use clinical whites with a calming teal accent
- **Use HSL thinking** — pick a hue, then vary saturation and lightness for primary/secondary/accent rather than picking unrelated hex codes
- **Test contrast mentally** — if your text color on your background color wouldn't pass a squint test, adjust
- **Preset colors are starting points, not boundaries** — if "Dark Professional" is close but the cyan feels wrong for your topic, shift it. Use `#2EADD3` instead of `#00D4FF`. The structure matters more than the exact hex values

### Example: Custom style for a jazz music presentation

| Property | Value |
|----------|-------|
| **Primary** | `1A1423` (smoky purple-black) |
| **Secondary** | `2D1F3D` (deep plum) |
| **Accent** | `D4A853` (brass gold) |
| **Text** | `F0E8D8` (warm cream) / `8A7E72` (muted) |
| **Header font** | Georgia |
| **Body font** | Calibri Light |
| **Shapes** | Rounded rectangles (0.15); warm gold 1pt borders; soft shadows |
| **Motif** | Brass-gold accent curves; warm spotlight-style generated imagery; intimate club atmosphere |
| **Backgrounds** | Smoky dark throughout; generated atmospheric images for title/closer |
| **Image prompts** | Add "jazz club atmosphere, warm spotlight, smoky, brass instruments, moody golden lighting, no text" |

## Style Blending

Sometimes the best design borrows from two styles. This is allowed — with discipline.

### Rules for blending

1. **Blend at most 2 styles** — more than that creates visual chaos
2. **Take the palette from one, the motif/shapes from the other** — don't mix palettes; that's where incoherence starts
3. **Both styles must share compatible tones** — you can blend Cinematic Minimal + Documentary (both dark and moody), but not 3D Clay + Constructivism (completely different energy)
4. **Declare what you're blending up front** — e.g., "Using Cinematic Minimal palette with Documentary motifs." This helps you stay consistent

### Good blends

| Blend | Why it works |
|-------|--------------|
| Cinematic Minimal palette + Documentary motifs | Both are dark and narrative-driven |
| Nordic Research structure + Pure Minimal aesthetics | Both value clarity and whitespace |
| Botanical colors + Wabi-Sabi shapes | Both are organic and earthy |
| Geometric Bold layout + Dark Professional palette | Both are confident and structured |

### Bad blends (avoid)

| Blend | Why it fails |
|-------|--------------|
| 3D Clay + Constructivism | Playful warmth clashes with angular propaganda |
| 8-bit Pixel + Morandi | Neon retro vs. muted sophistication |
| Comic + Pure Minimal | Maximum visual noise vs. maximum restraint |

## Applying a Style Consistently

When using a style preset, follow these rules:

1. **Use ALL properties** — don't just take the colors. Apply the fonts, shapes, motif, and background strategy together.

2. **Generate images in the style** — always append the style's image prompt modifier when using `generate_image`. This ensures visual consistency across all slides.

3. **Vary layouts within the style** — a style defines the visual language, not a single layout. Use different layout patterns (two-column, stat callout, image+text, grid) while keeping the style's colors, fonts, and motif consistent.

4. **Maintain the motif** — if the style says "accent bars on left edges," put them on every content slide, not just one. If it says "thin gold lines," use them throughout.

5. **Respect the background strategy** — each style specifies which slides get which backgrounds. Don't put dark backgrounds on a Pure Minimal deck or white backgrounds on a Cinematic Minimal deck.

6. **Blending is okay, chaos is not** — you may blend elements from up to 2 compatible styles (see [Style Blending](#style-blending)). But randomly mixing 3+ styles creates visual incoherence. When in doubt, commit to one.
